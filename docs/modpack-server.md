# Server
Can be used if you want to host the server yourself

The server uses everything from the [minimal](modpack-minimal.md) variant as well the following

**Quality of Life**
- [Mine Spawners](https://modrinth.com/mod/mine-spawners), allows you to remove spawners with a silk touch pickaxe
- [Universal Graves](https://modrinth.com/mod/universal-graves), customisable grave mod
- [Tree Harvester](https://modrinth.com/mod/tree-harvester), harvest trees instantly

**Miscellaneous**
- [No Prying Eyes](https://modrinth.com/mod/no-prying-eyes), removes telemetry
- [Simple Voice Chat](https://modrinth.com/mod/simple-voice-chat), proximity voice chat

**World**
- [BlueMap](https://modrinth.com/plugin/bluemap?loader=fabric&version=1.20.1), Minecraft mapping tool that adds 
  - [BlueMap Offline Player Markers](https://modrinth.com/mod/bluemap-offline-player-markers-(fabric)), adds a marker where players have logged off
  - [Create BlueMap](https://modrinth.com/mod/create-bluemap), adds support for Create to BlueMap
  - [OpenPaC BlueMap](https://modrinth.com/mod/openpac-bluemap-remapped)
- [Open Parties and Claims](https://modrinth.com/mod/open-parties-and-claims), allows players to claim chunks and make parties
- [YUNG's Better Caves](https://modrinth.com/mod/yungs-better-caves), overhauled cave generation
- [YUNG's Better Desert Temples](https://modrinth.com/mod/yungs-better-desert-temples), complete redesign of desert temples
- [YUNG's Better Dungeons](https://modrinth.com/mod/yungs-better-dungeons) complete redesign of dungeons
- [YUNG's Better End Island](https://modrinth.com/mod/yungs-better-end-island), overhaul of the main End Island where the dragon is fought
- [YUNG's Better Jungle Temples](https://modrinth.com/mod/yungs-better-jungle-temples), complete redesign of jungle temples
- [YUNG's Better Mineshafts](https://modrinth.com/mod/yungs-better-mineshafts)
- [YUNG's Better Nether Fortresses](https://modrinth.com/mod/yungs-better-nether-fortresses)
- [YUNG's Better Ocean Monuments](https://modrinth.com/mod/yungs-better-ocean-monuments), complete redesign of ocean monuments
- [YUNG's Better Strongholds](https://modrinth.com/mod/yungs-better-strongholds), complete redesign of strongholds
- [YUNG's Better Witch Huts](https://modrinth.com/mod/yungs-better-witch-huts), adds overhauled witch huts to swamps
- [YUNG's Bridges](https://modrinth.com/mod/yungs-bridges), adds naturally generated bridges throughout the world
- [YUNG's Cave Biomes](https://modrinth.com/mod/yungs-cave-biomes)
- [YUNG's Extras](https://modrinth.com/mod/yungs-extras), adds extra structures

**Dependencies**
- [Collective](https://modrinth.com/mod/collective), library for Tree Harvester
- [TerraBlender](https://modrinth.com/mod/terrablender), library for adding biomes
