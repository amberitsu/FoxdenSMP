# Full
This adds everything from the [minimal](modpack-minimal.md) and the [optimized](modpack-optimized.md) variants, as well as the following

**Interface**
- [Mod Menu](https://modrinth.com/mod/modmenu), Easily view and manage the mods you have installed
- [BetterF3](https://modrinth.com/mod/betterf3), replaces the default F3 HUG with a customizable and more readable HUD
- [Open Parties and Claims](https://modrinth.com/mod/open-parties-and-claims), adds a interface for managing and claiming chunks
- [VoxelMap-Updated](https://modrinth.com/mod/voxelmap-updated), adds a minimap and worldmap, which can show all parts of the world you have explored so far. As well as add the option to set waypoints
- [EMI](https://modrinth.com/mod/emi), extensive recipe and item viewer
  - [AE2 EMI Crafting Integration](https://modrinth.com/mod/ae2-emi-crafting), integrates EMI into Applied Energetics 2 terminals
  - [EMI: Create Schematics](https://modrinth.com/mod/emi-create-schematics), add schematic items from the Create mod to EMI

**Gameplay**
- [Zoomify](https://modrinth.com/mod/zoomify), a customizable zoom mod
- [LambDynamicLights](https://modrinth.com/mod/lambdynamiclights), adds dynamic lights without the need for more resource intesive shaders

**Quality of Life**
- [InvMove](https://modrinth.com/mod/invmove), allows you to still move around while accessing the inventory
- [Monsters in the Closet](https://modrinth.com/mod/monsters-in-the-closet), highlights mobs that are preventing you from sleeping

**Miscellaneous**
- [No Prying Eyes](https://modrinth.com/mod/no-prying-eyes), removes telemetry
- [Simple Voice Chat](https://modrinth.com/mod/simple-voice-chat), proximity voice chat
