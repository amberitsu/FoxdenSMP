# Optimized
Includes everything from the [minimal](modpack-optimized.md) variant. As well as the following optmization mods
- [Sodium](https://modrinth.com/mod/sodium), optimizes the Minecraft client to improve frame rate, as well as reduce micro-stutters
- [Sodium Extra](https://modrinth.com/mod/sodium-extra), adds extra options to Sodium
- [Lithium](https://modrinth.com/mod/lithium), optimizes game psyhics, mob AI, etc to improve performance
- [ModernFix](https://modrinth.com/mod/modernfix), improves performance, reduces memory usage and fixes many bugs that are present in MC
- [ImmediatelyFast](https://modrinth.com/mod/immediatelyfast), optimizes the way the GPU is utilized to improve performance
- [Memory Leak Fix](https://modrinth.com/mod/memoryleakfix), fixes memory leaks present in MC, in order to prevent the game from crashing once you run out of memory
- [FerriteCore](https://modrinth.com/mod/ferrite-core), optimizes memory usage
- [Entity Culling](https://modrinth.com/mod/entityculling), Uses asynchornous path-tracing to hide blocks and entities that aren't visible
