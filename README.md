# The Server
*The server works with a **whitelist**. If you want to get access to the server, you can either send a message in the minecraft server channel from [The FoxDen Discord](https://discord.gg/SEc8J9v2BY) or DM me @amberitsu!*

The modpack has a focus on things such as exploration, automation as well as a couple of QoL additions. 

Some of the key mods are
- Create
  - Inlcuding expansion mods such as `Create New Age`
- Farmer's Delight
  - Adds many different cooking systems, including some that integrate with Create
- Applied Energetics
  - Futuristic storage system with advanced options
- Tom's Simple Storage Mod
  - Early game/easier alternative to Applied Energetics
- Waystones
  - Craftable waypoints
- Spawners+
  - Craftable spawners
- Naturalist
- Simple Voice Chat
  - Proximity voice chat
- Blue Map
  - 3D web based world map
