# FoxdenSMP
## The Server
The MineCraft modpack has a focus on things such as exploration, automation as well as a couple of QoL additions.

Alongside this the server also works with a **whitelist**, so if you want to be added, you can tag me in this channel or DM me (@amberitsu).

## Modpack variants
- [Full](https://cdn.modrinth.com/data/OpjxbmTa/versions/sr65kvtA/Amber%20and%20friends%20SMP%20-%20Full.mrpack), recommended version, includes several QoL mods, optimization as well as voice chat
- [Optimized](https://cdn.modrinth.com/data/OpjxbmTa/versions/OSmEpuJu/Amber%20and%20Friends%20SMP%20-%20Optimized.mrpack), same as the minimal version, but includes some optimization mods to improve performance
- [Minimal](https://cdn.modrinth.com/data/OpjxbmTa/versions/qiT1Ho74/Amber%20and%20Friends%20SMP%20-%20Minimal.mrpack), barebones version of the mod, only includes the minimum to connect to the server
- [Server](https://cdn.modrinth.com/data/OpjxbmTa/versions/scVY10Ju/Amber%20and%20Friends%20SMP%20-%20Server.mrpack), in case you want to host your own version of the modpack
- [Singleplayer](https://cdn.modrinth.com/data/OpjxbmTa/versions/9j8FyhXz/Amber%20and%20Friends%20SMP%20-%20Singleplayer.mrpack), modpack optimized for a consistent singleplayer experience, includes many of the QoL mods from the full version as well as some new ones. Alongside the optimizations
